# Intrusion Detection System

A machine learning based intrusion detection system for detecting cyber attacks using the NSL-KDD dataset.

## Features
- Data preprocessing
- Feature selection
- Attack classification
- Confidence calibration
- Explainable AI using SHAP
- Model evaluation and visualization

## Technologies
- Python
- PyTorch
- Scikit-learn
- Pandas
- NumPy
- SHAP
- Matplotlib

## Dataset
NSL-KDD Dataset

## Run
pip install -r requirements.txt
python main.py
